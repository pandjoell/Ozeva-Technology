== Installation ==

**How to Add a Google Map to Your WordPress Website - Automatic Installation:**
1. Go to your WordPress admin dashboard
2. Navigate to Plugins > Add New
3. Search for "Super Simple Map Embeds"
4. Click "Install Now" and then "Activate"

**Manual WordPress Plugin Installation:**
1. Download the plugin files
2. Upload to `/wp-content/plugins/super-simple-map-embeds/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

**Getting Started - Embed a Google Map in Minutes:**
1. Create or edit any post or page in WordPress
2. Click the '+' button to add a new block
3. Search for "Super Simple Map" and select it
4. Enter your address or location in the block settings
5. Customize zoom level and map height as needed
6. Your embedded Google Map is ready!

== Frequently Asked Questions ==

= Do I need a Google Maps API key for this WordPress plugin? =

No! This Google Map plugin uses Google's public embed API, so no API key, registration, or billing setup is required. Just install this WordPress plugin and start adding maps immediately.

= Can I customize the Google Map appearance? =

Yes! You can adjust the zoom level (1-20) and map height (200-800px) directly in the block settings. The maps are also fully responsive and adapt to your WordPress template styling.

= Will this Google Map plugin slow down my WordPress website? =

Not at all! This WordPress plugin is extremely lightweight and only loads Google's embed code when needed. Maps are loaded efficiently and won't impact your site's performance.

= Can I add multiple locations with this map plugin? =

Absolutely! You can add as many map blocks as you need to any post or page. Each embedded Google Map can have different locations and settings for multiple locations display.

= Does this WordPress Google Map plugin work with my template? =

Yes! This Google Map plugin works with any properly coded WordPress template. The maps are responsive and will adapt to your WordPress website's layout and styling.

= Can I use this plugin for a store locator? =

Perfect for store locator functionality! Add maps to contact pages, store locator pages, office locations, event venues, or any location-based content on your WordPress website.

= Does this work as a WordPress widget? =

Yes! This plugin supports WordPress widget functionality, allowing you to add Google Maps to sidebars and widget areas throughout your WordPress website.

= Can I use shortcodes with this Google Map plugin? =

While primarily designed as a Gutenberg block, this WordPress plugin is compatible with shortcode implementations for advanced users who prefer that method.
